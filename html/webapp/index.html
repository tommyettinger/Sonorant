Twirly Swirly Noise Viewer

Controls:

  • 'f' decreases frequency, which mostly affects the size of details (you can use Shift here, also).
  • 'd' changes the number of divisions in the rotational symmetry; odd numbers spiral, even numbers mirror half of their divisions.
  • 'n' changes noise type, which affects the shapes and sizes that appear.
  • 'r' changes the fractal type between 4 different modes (two often look similar).
  • 'i' changes the interpolator used to choose how much dark vs. light to use (there are many of these).
  • 'c' randomizes the main hue.
  • 'v' increases the variation between hues used for dark and light colors (with Shift, it decreases variation).
  • 's' moves the seed up one.
  • 'e' moves the seed down one, to an earlier seed.
  • '/' randomizes the seed.
  • 'h' adds higher detail by increasing the octave count.
  • 'l' makes detail lower by decreasing the octave count.
  • 'w' copies a summary to your clipboard that can be used to load the same noise on the desktop version.
  • On desktop, 'p' will paste in a summary from the clipboard and try to load noise based on it. This does not work on the web...

For almost all of these that change a variable to the next in a sequence, you can hold Shift while pressing the key to go to the previous item in the sequence instead. The same is true for variables that get larger when you press a key; holding shift will make the variable smaller.
